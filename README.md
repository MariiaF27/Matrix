# Лабораторная работа: Непрямоугольные матрицы

## Задание

Для непрямоугольной матрицы целых чисел найти вектор `b`, где `b[i]` = сумма цифр всех элементов i-й строки.

## Входные данные

- Число строк `m`
- Для каждой строки: `n` (длина строки) и `n` целых чисел

## Выходные данные

- Исходная матрица
- Вектор результатов

## Требования

- Запрещены глобальные переменные
- Проверка корректности ввода
- Разбиение на `.c` и `.h` файлы
- Проверка памяти (valgrind)

## Дополнительное задание

Хранение матриц в бинарном файле произвольного доступа:
- Элементы только во внешней памяти
- Информация о строках — в заголовке файла
- Имена файлов вводятся пользователем


## Сборка и запуск

# Основная программа
gcc -Wall -Wextra -Werror main.c matrix.c -o matrix

# С доп. заданием
gcc -Wall -Wextra -Werror main.c matrix.c file_matrix.c -o matrix_file

./matrix
